Books Dataset

All content/data/visualisations on this website is based on the 'Goodreads-books' dataset on Kaggle: https://www.kaggle.com/datasets/jealousleopard/goodreadsbooks
This data was taken entirely from the Goodreads API

The goal of this website is to present the data from this dataset in several different ways, allowing you to easily navigate it


This website has several methods for navigating this dataset:

  • Using the search bar on the navigation bar (Please note your search term must be exact and case-sensitive)
  • Using the full dataset page to view the entire dataset
  • By looking at the visualisations/graphs of some parts of the data below
